STL DOM Interface

This is the standard dom handling package for LovelyIO

Usage

It is mostly the good old RightJS except there is just one $ jquery-like function

Lovely(['dom'], function($) {
  $('#my-id')
    .html('<p>trololo</p>')
    .parent().addClass('parent');
});

See the API docs for the actual usage information
