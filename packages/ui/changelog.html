Ui Changelog

October 12, 2012 - Nikolay Nemshilov

  • Version 2.1.2
  • Module intiialization fix

October 2, 2012 - Nikolay Nemshilov

  • Version 2.1.1

  • UI.Spinner was reworked and now provides a fancy CSS3 based circular spinners by default. The old flat ones are also available through options

    new UI.Spinner();             // -> circular spinner
new UI.Spinner(type: 'flat'); // -> the old style flat spinner

  • UI.Options mixin module was added. unlike the core.Options mixin, this one allows you to specify widget and HTML options in one hash, plus it reads the data-widget-name attributes from dom elements

    MyWidget = Element.inherit
  include: UI.Options
  extend:
    Options: # global options
      something: 'something'

  constructor: (options)->
    options = @setOptions(options)

    super 'div', options

my_widget = new MyWidget(id: 'my-id', something: 'new something')

  • UI.Menu unit #show and #hide methods now can accept the usual fx-name arguments

  • Standard lovely.io testing coverage

  • FF font assets serving fixes

  • Package dependencies update

July 2, 2012 - Nikolay Nemshilov

  • Version 2.0.1

  • UI.Modal overlap option was added

    var m1 = new UI.Modal();
var m2 = new UI.Modal({overlap: true});

m1.show(); // shows the first modal
m2.show(); // doesn't hide the first modal automatically and shows the new one over it

  • Getting rid of the '.lui-aligner' element in favor of the :before generated element in the UI.Locker and UI.Modal widgets

June 10, 2012 - Nikolay Nemshilov

  • Version 2.0.0
  • FontAwesome is the default source of icons

  • New buttons/icons realations

    <a href="" class="lui-button lui-icon-ok">Okay</a>

    This makes an embedded into the button icon with gray background

    <a href="" class="lui-button"><i class="lui-icon-ok"></i>Okay</a>

    This makes the icon to be inlined in the button along with the caption text

  • The old button-like icons were moved to the buttons functionality, and UI.Icon now just an icon, without any borders or backgrounds

  • Button groups added

    <div class="lui-button-group">
  <a href="" class="lui-button"><i class="lui-icon-ok"></i></a>
  <a href="" class="lui-button"><i class="lui-icon-remove"></i></a>
</div>

  • Menu headers support added

    <nav class="lui-menu">
  <h4>Beverages</h4>
  <a href="#">Coffee</a>
  <a href="#">Green Tea</a>
  <a href="#">Orange Juice</a>

  <h4>Starters</h4>
  <a href="#">Crap salad</a>
  <a href="#">Crab soup</a>
</nav>

  • Dropdown menus for the buttons were added

    <a href="#" class="lui-button lui-button-dropdown" data-toggle="#menu-1">Profile</a>
<nav id="menu-1" class="lui-menu">
  <a href="#" class="lui-icon-home">Home</a>
  <a href='#' class="lui-icon-cog">Settings</a>
  <hr>
  <a href="#" class="lui-icon-signout">Sign Out</a>
</nav>

  • UI.Icon class API was changed, now you can create icons just by name

    var icon = new UI.Icon('edit');
