Glyph-Icons

This little package provides the glyphicons drop in styles.

Usage

Just hook up this package in any standard way on your page and it will automatically add icons to your links with classes add, edit, delete, etc.

<html>
  <head>
    <script src="http://cdn.lovely.io/core-1.1.0.js"></script>
    <script type="text/javascript">
      Lovely(['glyph-icons-1.0.2'], function() {})
    </script>
  </head>
  <body>
    <a href="..." class="new">Create new</a>
    <a href="..." class="edit">Edit</a>
    <a href="..." class="delete">Delete</a>
    <a href="..." class="search">Search</a>
  </body>
</html>

See the live demo page for the complete list of supported classes
