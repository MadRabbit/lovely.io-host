Analog Clock

AnalogClock is a simple package that provides a fully CSS customisable analog clock widget

Usage

<html>
  <head>
    <script src="http://cdn.lovely.io/core-1.2.0.js"></script>
    <script type="text/javascript">
      Lovely(['analog-clock'], function(AnalogClock) {
        new AnalogClock().insertTo(document.body);
      });
    </script>
  </head>
</html>
