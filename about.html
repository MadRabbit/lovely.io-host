About Project

Lovely.IO is a front-side development platform, which includes a modular framework, development tools, CDN hosting, packaging and asynchronous modules loading with automatic dependencies handling.

The idea of the project is to provide a full-stack environment for creating easy to maintain and use front-side modules. In a sense it is much like server-side languages package managers, but for the front side.
