Content Delivery Speed Test

Disclamer: Lovely.IO is AMD based and loads 4 files by using the following layout 

<html>
  <head>
    <script src="http://cdn.lovely.io/core-1.4.1.js"></script>
    <script>
      Lovely(['dom-1.4.3', 'ajax-1.1.4', 'fx-1.1.1'], function() {});
    </script>
  </head>
</html>

For jQuery delivery we use the googleapis hosting with the following layout 

<html>
  <head>
    <script src="http://ajax.googleapis.com/ajax/libs/jquery/2.0.0/jquery.min.js"></script>
  </head>
</html>
